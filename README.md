**Does ConfAI send my data to a server?**
No. The application runs locally on your machine. Your configuration files stay on your hard drive. 

**What happens if my AI agent updates?**
ConfAI detects changes in the agent software. It checks your configuration to see if new settings appear. It asks if you want to add these new settings to your file.

**Can I run multiple agents at once?**
Yes. ConfAI manages the files for each agent individually. It does not interfere with the programs themselves.

**Will this software break my existing code?**
ConfAI modifies configuration files only. It does not touch your source code. The backup system ensures you always have a copy of your previous working state.

**Do I need a paid license?**
